# Application Features

Here are the main features of **Skinscanner**:

-   **Price Aggregation**: Collects skin price data from various sources and marketplaces in real-time.

-   **Price Comparison**: Displays a price comparison from different marketplaces for the same skin item, allowing users to easily find the best deals.

-   **Advanced Search**: Users can search for specific skins by name, game, or other categories.

-   **Filter and Sort**: Provides options to filter search results (e.g., by price, popularity) and sort them as needed.

-   **Intuitive User Interface**: A clean and easy-to-use interface to ensure a smooth user experience.

-   **Item Details**: Displays detailed information for each skin, including images, descriptions, and price history if available.
